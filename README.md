# mt_reverse
Mersenne Twisterの
> 十分な長さの出力列を見れば、その後の数列を完全に予言することができます。
> (http://www.math.sci.hiroshima-u.ac.jp/~m-mat/MT/faq.html)

という特徴をテストする為のコードです。
